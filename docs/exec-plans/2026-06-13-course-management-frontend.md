# 执行计划：课程目录与开课安排前端

日期：2026-06-13  
状态：已完成

## 目标

将现有课程目录与开课安排后端能力接入前端，形成管理员创建、管理员和教师查询的完整流程。

## 完成内容

- 课程管理页使用标签切换课程目录与开课安排。
- 课程目录支持名称或编号搜索、状态筛选、创建和详情查看。
- 开课安排支持课程、班级、学年、学期筛选，以及创建和详情查看。
- 创建开课安排使用启用课程、班级和启用教师下拉选择。
- 学生角色不开放课程管理入口，仍通过独立课程表查看本人课程。
- 未修改后端接口和数据库结构。

- 前端 `npm run build`：通过。
- Playwright 模拟 API 流程：课程创建、课程详情、开课安排创建和详情均通过。
