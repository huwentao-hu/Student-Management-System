# 执行计划：班级管理与学生分班历史前端

日期：2026-06-13  
状态：已完成

## 目标

实现班级查询、创建、详情和学生分班历史管理前端，并遵循现有后端权限。

## 完成内容

- 管理员和教师可搜索、按入学年份筛选并查看班级详情。
- 管理员可创建班级并指定班主任教师账号 ID。
- 学生详情展示当前班级和完整分班历史。
- 管理员可在学生详情执行入班、转班和离班。
- 教师可查看学生分班历史，但不显示分班操作。

## 验证结果

- `npm run lint`：通过。
- `npm run build`：通过。
- Playwright 模拟 API 流程：班级创建、详情、入班、历史和离班均通过。

## 后续依赖

- 后端需要提供教师账号列表接口，前端才能将班主任 ID 输入改为教师下拉选择。
- 后端需要提供按班级查询学生名单接口，班级详情才能展示花名册。
