# 执行计划：学生新增、详情与编辑前端

日期：2026-06-13  
状态：已完成

## 目标

在现有学生列表基础上，完成学生新增、详情查看和编辑流程，并遵循后端角色权限。

## 完成内容

- 管理员可从学生列表进入新增页面并创建学生档案。
- 管理员和教师可查看学生详情。
- 管理员可编辑学生资料和状态，学号保持只读。
- 表单支持必填姓名、邮箱格式、出生日期和字段长度等浏览器校验。
- 空选填字段提交为 `null`，与后端请求模型一致。
- 页面支持桌面端和移动端布局。

## 验证结果

- `npm run lint`：通过。
- `npm run build`：通过。
- Playwright 模拟 API 浏览器流程：登录、列表、详情、编辑和新增均通过。
