# 执行计划：成绩管理

日期：2026-06-12  
状态：已完成

## 目标

实现百分制成绩录入、修改和角色范围查询。

## 执行步骤

1. 新增成绩表 Flyway 迁移。
2. 实现成绩录入或修改接口。
3. 校验学生在学期内修读对应开课安排。
4. 实现管理员、授课教师和学生的查询权限。
5. 补充集成测试并验证真实 PostgreSQL。
6. 更新所有相关文档。

## 验收条件

- 管理员和授课教师可为已修读学生登记成绩。
- 授课教师不能管理其他教师负责的成绩。
- 学生只能查看自己的成绩。
- 同一学生同一开课安排只有一条成绩。
- 自动化测试和真实 PostgreSQL 验证通过。

## 完成内容

- 新增 Flyway V8 成绩表迁移。
- 新增成绩首次录入和修改接口。
- 新增成绩单条查询和分页筛选接口。
- 实现学生修读关系校验。
- 实现管理员、授课教师和学生的数据范围权限。
- 抽取统一学期日期计算供课程表与成绩校验复用。
