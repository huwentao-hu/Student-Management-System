# 统计查询性能优化设计

状态：已实现  
最后更新：2026-06-12

## 目标

将成绩与考勤统计从加载全部实体后在 Java 中聚合，改为数据库聚合查询，减少内存占用和关联查询次数。

## 实现规则

- 使用 Spring Data JPA 聚合投影返回统计原始值。
- 数据库执行 `COUNT`、`AVG`、`MIN`、`MAX`、`SUM` 和条件计数。
- Service 继续负责权限、空统计归一化、比例计算和两位小数舍入。
- 不修改现有统计接口、响应字段或业务口径。
- 不新增统计表，不新增 Flyway 迁移。

## 验收条件

- 成绩教学班统计和学生学期汇总结果不变。
- 考勤教学班统计和学生学期汇总结果不变。
- 统计查询不再加载完整成绩或考勤实体集合。
- H2 与真实 PostgreSQL 验证通过。
