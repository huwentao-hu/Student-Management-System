# 学生课程表设计

状态：已实现  
最后更新：2026-06-12

## 目标

根据学生分班历史和班级开课安排，查询学生在指定学年学期的课程表。

## 学期日期规则

- 第一学期 `FIRST`：学年起始年份的 9 月 1 日至次年 1 月 31 日。
- 第二学期 `SECOND`：次年 2 月 1 日至次年 7 月 31 日。

学生在学期日期范围内曾归属的所有班级都会用于查询课程表。这样学生发生转班后，历史课程信息不会丢失。

分班历史已保证日期区间不重叠，因此课程表不会因异常重叠记录错误合并多个班级。

## API

`GET /api/students/{studentId}/timetable?academicYear=2026&semester=FIRST`

响应包含学生 ID、学年、学期、相关班级 ID 和开课安排列表。

## 权限

- 管理员：查询任意学生课程表。
- 教师：查询任意学生课程表。
- 学生：只能查询自己的课程表。

## 特殊情况

- 学生在该学期没有任何分班记录时，返回空课程表。
- 学生不存在时返回 `404 Not Found`。
- 学年或学期参数无效时返回 `400 Bad Request`。
