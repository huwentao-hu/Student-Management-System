# 考勤统计设计

状态：已实现  
最后更新：2026-06-12

## 目标

基于已创建课堂场次和已登记考勤记录，提供教学班考勤统计和学生学期考勤汇总，不新增统计数据表。

## 统计规则

- 出勤率公式为 `(出勤记录数 + 迟到记录数) / 已登记记录数 × 100%`。
- 请假和缺勤不计入出勤。
- 未登记考勤的学生不会被自动视为缺勤。
- 出勤率保留两位小数。
- 没有已登记记录时，所有数量字段返回 `0`，出勤率返回 `null`。

## 权限

- 管理员可查询所有教学班和学生的考勤统计。
- 授课教师只能查询自己负责教学班的考勤统计。
- 学生不能查询教学班统计，只能查询自己的学期考勤汇总。
- 教师首版不能查询学生完整学期汇总，避免查看其他教师课程的考勤。

## API

### 教学班考勤统计

`GET /api/attendance-statistics/course-offerings/{courseOfferingId}`

返回课堂场次数、已登记记录数、四种状态数量和出勤率。

### 学生学期考勤汇总

`GET /api/attendance-statistics/students/{studentId}?academicYear=2026&semester=FIRST`

返回学生指定学期的已登记记录数、四种状态数量和出勤率。

## 数据实现

统计基于 `attendance_sessions`、`attendance_records` 和 `course_offerings` 实时计算。数据库使用聚合投影直接计算记录数量和各状态条件计数，不加载完整考勤实体集合。考勤修改后统计结果立即变化。
