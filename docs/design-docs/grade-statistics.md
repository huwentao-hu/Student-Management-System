# 成绩统计设计

状态：已实现  
最后更新：2026-06-12

## 目标

基于已录入成绩提供教学班成绩统计和学生学期成绩汇总，不新增统计数据表。

## 统计规则

- 及格线固定为 `60.0` 分。
- 平均分、学分加权平均分和及格率保留两位小数。
- 教学班没有成绩时，数量字段返回 `0`，平均分、最高分、最低分和及格率返回 `null`。
- 学生指定学期没有成绩时，数量和总学分返回 `0`，学分加权平均分返回 `null`。
- 学生学期加权平均分公式为 `成绩乘课程学分之和 / 课程学分之和`。

## 权限

- 管理员可查询所有教学班和学生的成绩统计。
- 授课教师只能查询自己负责教学班的成绩统计。
- 学生不能查询教学班统计，只能查询自己的学期成绩汇总。
- 教师首版不能查询学生完整学期汇总，避免查看其他教师课程的成绩。

## API

### 教学班成绩统计

`GET /api/grade-statistics/course-offerings/{courseOfferingId}`

返回已录入成绩数、平均分、最高分、最低分、及格人数、不及格人数和及格率。

### 学生学期成绩汇总

`GET /api/grade-statistics/students/{studentId}?academicYear=2026&semester=FIRST`

返回成绩数、总学分、学分加权平均分、及格课程数和不及格课程数。

## 数据实现

统计基于 `grades`、`course_offerings` 和 `courses` 实时计算。数据库使用聚合投影直接计算数量、平均分、极值、学分和加权总分，不加载完整成绩实体集合。系统不保存统计快照，因此成绩修改后统计结果立即变化。
