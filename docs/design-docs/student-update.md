# 学生信息修改设计

状态：已实现  
最后更新：2026-06-12

## 目标

允许管理员修改学生档案，同时保证系统生成的学号不可修改。

## API

`PUT /api/students/{id}`

仅管理员可调用。请求必须提供完整的可修改学生信息和状态：

```json
{
  "name": "张三",
  "gender": "MALE",
  "dateOfBirth": "2008-05-12",
  "phone": "13800000000",
  "email": "zhangsan@example.com",
  "enrollmentDate": "2026-09-01",
  "status": "ACTIVE"
}
```

## 规则

- 学号不属于修改请求字段。
- 修改入学日期不会改变学号。
- 管理员可修改学生信息和状态。
- 教师和学生不可修改学生档案。
- 学生不存在时返回 `404 Not Found`。

## 数据库影响

本功能未修改数据库结构。
